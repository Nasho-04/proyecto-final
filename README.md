# React + Vite

Bueno, la idea del proyecto final fue hacer una aplicación similar a "Slack" como propuso el profesor. Mi idea fue hacer las funcionalidades básicas dentro de todo parecidas, como lo son el tener una lista de entornos y canales, poder agregar entornos y canales y, por último, poder mandar mensajes. Luego de esto, agregué el LocalStorage, que se me complicó por no haberlo hecho desde el principio ya que tuve que cambiar varias bases de lo que ya tenía hecho. Por último decidí agregar unos botones con los cuales poder eliminar tanto entornos como canales, pero al notar que podía tocar el botón por accidente y eliminar algo que no quería, decidí agregarles una confirmación de eliminación y de esta manera solucionar ese problema. Por últimmo, aclarar que no es un clon de Slack en sí, sino que me enfoqué en sus funcionalidades básicas y le creé un estilo distinto, más colorido, aunque el diseño no sea lo mío.
Los desafío más importantes fueron, el LocalStorage por lo que mencioné previamente y el responsive design porque había cosas que al verlas desde la compu con el tamaño del celular se veían bien, pero, al abrirlo en el celular, cambiaban un poco. También se complicó un poco el hacer el menú desplegable de canales y que cambie según las medidas sin problemas, pero la verdad que disfruté el desafío y la pasé bien resolviendo cada problema que se me presentó.
Estoy muy contento con lo que logré y no se me ocurren más cosas para agregarle, si el hecho de aprobar o no depende de que haya faltado una pequeña funcionalidad y hay que agregarla, no tengo problema en hacerlo.

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh
